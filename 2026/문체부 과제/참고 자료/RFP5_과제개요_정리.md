# [문체부] 2026 PSR 기반 맞춤형 운동 추천 및 효과성 분석 AI 개발 (RFP5) - 과제 개요 정리

## 1. 과제 개요

- **과제명**: PSR 기반 맞춤형 운동 추천 및 효과성 분석 AI 개발
- **목표**: PSR 기반 신체 기능 이상 징후를 추론하고, 개인별 운동 루틴을 자동 생성하는 AI 모델 개발 → 운동 효과성을 과학적으로 검증
- **핵심 기술**: 운동 궤적·속도·심박·파워 등의 데이터를 활용해 가동성 및 대사 활성도 추론 정확도 **90% 이상** 달성
- **연계 기술**: 스포츠 지식 온톨로지와 LLM 기반 API 연계를 통해 적응형 운동 가이드 제공
- **실증**: 리빙랩 및 임상실증을 통한 실제 사용자 기반 효과성 검증

---

## 2. 필요성

- **비침습적 신체 분석 기술의 시급성**: 병원 진단 장비 없이 스포츠 현장 데이터(동작, 부하, 회복률)만으로 신체적 이상 징후 조기 발견 기술 필요
- **개인화된 스포츠 AX 구현**: 파편화된 운동 정보를 통합하여 사용자의 실제 신체 능력에 최적화된 운동 강도와 종목 매칭 → 고도화된 추론 엔진 필수

---

## 3. 연구 내용

### 3-1. PSR 기반 신체 기능 정밀 분석 기술
- **(개념 연구)** 실내 종목별 PSR 데이터 기반 효과성(Performance Gain) 측정 지표 정의
- **(역학적 분석)** 운동 궤적 및 속도 변이 데이터를 활용한 근골격계 불안정성 및 좌우 불균형 정밀 추론 모델 개발
- **(생리적 분석)** 운동 부하 대비 심박 회복률 및 파워 출력 지속성을 연계한 대사적 활성도 추론 알고리즘 구축

### 3-2. 맞춤형 운동 최적화 가이드 알고리즘
- 분석된 신체 기능 바탕으로 부상 위험 최소화 + 운동 효과 극대화 → **개인 맞춤형 운동 루틴 자동 생성**
- 사용자 피드백(RPE, 인지 부하 등)을 학습하여 실시간 운동 강도 조정 → **적응형(Adaptive) 가이드라인 모델** 구현

### 3-3. 스포츠 지식 기반 추론 엔진 고도화
- 운동역학, 생리학 등 스포츠 전문 지식(Ontology)을 AI 모델에 주입 → 추론 결과의 과학적 근거 확보
- 연계과제의 LLM 에이전트와 연동 가능한 API 구조 설계 및 데이터 상호운용성 확보

### 3-4. 과제 간 연계 및 총괄협의체 참여
- 협의체 및 총괄과제 책임자의 변경/조정 요구사항 반영 필수
- 총괄과제의 데이터 표준화 및 연계·활용을 위한 인터페이스와 평가 기준 협의

## 5. PSR(Personal Sports Record) 정의

- **정의**: 개인이 평생에 걸쳐 수행한 모든 스포츠 및 신체활동 데이터를 통합적으로 수집, 관리, 활용하는 기술 및 서비스
- **핵심 특징**:
  - **생애주기 데이터 관리**: 운동 시작 시점부터 현재까지 전 생애주기 운동 기록 누적 관리
  - **통합형 데이터**: 훈련 강도, 빈도, 운동 종목, 신체 변화(심박수, 칼로리 소모 등) 다각적 데이터 통합
  - **마이데이터/스마트 헬스케어**: 개인 운동 데이터를 스스로 소유·활용, 코치/병원 등 맞춤형 서비스에 활용
  - **성장 기록**: 데이터 분석을 통해 개인 성장 시각화, 운동 목표 달성 지원

### 데이터 수집
| 수집 장비 | 수집 데이터 |
|-----------|------------|
| Smart Watch | 신체활동(균형, 보행), 피로/통증 |
| 패치(S-Patch) | 심박 회복률, 파워 출력 지속성 (생리적 분석 지표) |
| Bio Armour | 관절 가동범위(ROM), 근력(Arm Curl, STS) (역학적 분석 지표) |
| Mobile Software | 사용자 인터페이스, 데이터 전송 |

### AI 분석 → 추천
- 근골격계 불안정성 및 좌우 불균형 정밀 추론 모델
- 대사적 활성도 추론 알고리즘
- **출력**: 개인 맞춤형 운동 가이드 프로그램 / AI 기반 건강 상태 평가 모델 / 사용자 피드백 기반 지속학습형 AI 추천 시스템

## 8. 성능지표

| # | 성능지표 | 단위 | 1차년도 | 2차년도 | 3차년도 |
|---|---------|------|---------|---------|---------|
| 1 | PSR 과제 상호운용 API (운영 API 수/호출성공률) | ea/% | 2/80% | 4/85% | 6/95% |
| 2 | 운동 처방 일치도 (ICC) | % | 0.8 | 0.85 | 0.9 |
| 3 | 데이터 가용성 | % | 70 | 85 | 95 |
| 4 | 사용성평가(소비자/전문가) 만족도 | % | - | 85 | 90 |
| 5 | AI 머신러닝 분류성능평가 | 건 | - | - | 1 (ISO/IEC TS 4213) |
| 6 | AI 데이터 품질평가 | 건 | - | - | 1 (ISO/IEC 5259-2) |

### 성능지표 기준
1. AI 추론과정에 직접 호출하여 사용하는 외부 지식 도구(API, MCP, DB, Ontology 등)의 개수와 상호호출
2. 의료전문가 검증셋(Doctor-Verified Set) 자체 구축 및 총괄협의체 협의 조정 후 사용
3. 스포츠 특화 LLM 및 개인운동기록 융합서비스 실증과제에서의 서비스 호출 성공률 기반 실시간 서비스 응답성 지표
4. 실제 사용자(운동참가자, 코치, 일반사용자 등) 대상 만족도 수집, 만족도 지표는 총괄협의체에서 제시
5. 국제표준(ISO/IEC TS 4213)에 따른 데이터(위치, 동작추적, 재현, 인지, 반응 등) 및 환경조건에서의 성능평가
6. 국제표준(ISO/IEC 5259-2)에 따른 품질 모델 및 품질 측정 기준 평가
